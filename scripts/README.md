# Computational Workflow Scripts

This directory contains representative scripts used for deep-time slab reconstruction, mantle density model preparation, geoid analysis, and scientific visualization.

The workflow integrates:
- GPlates / PyGPlates reconstruction
- Slab geometry generation
- Mantle density heterogeneity modelling
- HC mantle circulation workflows
- GMT-based geoid visualization
- Python and MATLAB post-processing

## Directory Structure

### reconstruction/
Scripts related to:
- Subduction zone extraction
- Plate reconstruction
- Slab geometry generation

### density_models/
Scripts used to:
- Generate global slab grids
- Prepare density heterogeneity inputs
- Convert slab structures into HC-compatible formats

### geoid_analysis/
Scripts used for:
- Geoid comparison
- Correlation analysis
- Viscosity sensitivity testing

### visualization/
Scripts for:
- GMT map generation
- Figure production
- Reconstruction visualization

### utilities/
General helper functions and coordinate-processing tools

## Notes

The repository contains representative workflow scripts intended to demonstrate the computational methodology.

Some production workflows and unpublished datasets are not included due to ongoing research and publication considerations.
